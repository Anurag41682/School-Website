# School Website

## Instruction to set up the project locally :

```sh

# to clone the repository locally
git clone https://github.com/Anurag41682/School-Website.git

# to install the dependency
npm install

# to start the server
npm run dev


```

## Technologies Used :

- HTML5
- CSS3
- Javascript
- React
- TailwindCSS
- ShadCN
